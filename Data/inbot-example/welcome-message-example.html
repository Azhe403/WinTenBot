🏗 Command Pattern /welcome message text_message 📃 Example Message: /welcome message Hai {$new_members}, selamat {$time_current} Selamat datang di Kontrakan {$chat_title} Jumlah anggota sekarang {$member_count} ✍ Variable yang tersedia $chat_title : Nama grup, $new_members : Anggota baru dengan username, $new_bots : Anggota baru berupa bot, $no_username : Anggota baru tanpa username, $new_members_count : Jumlah anggota baru, $new_bots_count : Jumlah anggota baru berupa bot, $no_username_count : Jumlah anggota baru tanpa username, $time_current : Ex. Selamat pagi, siang, .., $member_count : Jumlah anggota setelah anggota baru masuk, ⚠ Warning!: Salah ketik variable dapat mengakibatkan Welcome Message tidak terkirim! 📃 Note: Ketika kamu mematikan Unified Welcome Message, kamu harus mengatur pesan welcome untuk setiap anggota. Unified Welcome Message enabled by default.
